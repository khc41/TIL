## 📌 TIL로 성장하는 법
- 전문가와의 간극을 좁히려면 꾸준한 학습과 실천이 중요하지만, 유효한 피드백을 통해 제대로 성장하고 있는지 점검하는 것도 필수적이라 생각했다.
- 이를 위해 현실적으로 가장 효과적인 방법은 기술 블로그를 읽고 정리하거나, 컨퍼런스 영상을 보고 기록하는 것이라 판단했다.
- 이를 통해 업계 전문가들이 실무에서 겪는 문제와 해결 방법을 배우며 성장하고자 한다.
